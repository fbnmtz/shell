# Shell

This repository contains some scripts and a simple library system.

## Install lib/scripts

```bash
# clone this repo to your '~/home'
 git clone https://github.com/fbnmtz/shell ~/bin
```

## Using libs

```bash
# import lib system to your script
source ~/bin/lib/init  

# load basic lib 'args'
use args
```

### Main Lib's

* init
* args
* utils
* hashs


### Lib 'args'

* main functions
  * requirements
  * xargs
  * xrun
